Framework used for this site

I originally developed this site using pure HTML5, CSS3, and ECMAScript6 and hosted it on ApolloHosting.com in early February 2015 just before my Junior year Spring semester career fair at UD. Recently, I transformed the original code to Node.js with bootstrap ran on Amazon Web Services' BeanStalk. This was done so that I could do some server-side processing on my projects portfolio page, which is not made easy with ApolloHosting.
